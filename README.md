# Data-Structures-in-JAVA
Data Structures completely fuctional and programed in JAVA

* QUEUE
* STACK
* MINHEAP
* BINARY TREE
* RED-BLACK TREE
* AVL TREE
* MAP (DICTIONARY) 
* GRAPHS

* QUICKSORT
* MERGESORT
* GENERIC HASH
* DJIKSTRA
